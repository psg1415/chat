# 채팅프로그램

* node.js를 이용한 채팅창 [사이트링크](https://psgchat.herokuapp.com)

### 구현기능

    * socket.io를 이용한 실시간 채팅
    * css로 만든 다크모드와 라이트모드
    * viewport를 이용한 모바일사이즈일때 폰트사이즈 변경
    * moment를 사용하여 채팅전송시간 표시
